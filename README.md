# GenAI InfoSite

My Pre-Finals static website about Generative AI (GenAI), its FAQs, ethical use, and an analogy comparing AI-assisted work in development.  

The website uses is coded in html, with the help Bootstrap 5, a little bit of Chatgpt for things I dont know how to code, and a  CSS file for styling.

---

Project Overview

This website is designed to educate beginners about:

- What Generative AI is  
- How it works  
- Examples of GenAI tools  
- Strengths and limitations of GenAI  
- Responsible and ethical use of AI  
- A practical analogy on AI-assisted development work

How to Run the Project

1. Clone or download the repository.  
2. Ensure the folder structure is intact so CSS loads properly.  
3. Open any HTML file (e.g., `index.html`) in a web browser.  

No installation is required; it is a static website.
